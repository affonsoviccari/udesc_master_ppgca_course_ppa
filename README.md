# Course XYZ - X Y Z
Estrutura de pastas padrão para organização de disciplinas.

## Estrutura
```
course_xyz/
├── 1_classes/
├── 2_materials/
└── 3_assignments/
```

## Diretórios
- **1_classes**: Aulas, anotações, resumos, slides próprios.
- **2_materials**: Materiais do professor, slides, artigos, referências.
- **3_assignments**: Trabalhos, exercícios, listas, projetos, relatórios.

---
> Aplique o repositório `template_course_xyz` como template para utilizar esse padrão organizacional.
